# TradeUI

A simple TradingSimulation in C++ using ImGUi. 
This is currently work in Progress and supposed to become the UI "Extension" for this: https://github.com/LeonardFoerster/hft-engine.git

## Deployment

To deploy this project run

```bash
  git clone https://github.com/LeonardFoerster/TradeUI
  cd TradeUI
  cmake --build build
  ./build/TradingDashboard
```
